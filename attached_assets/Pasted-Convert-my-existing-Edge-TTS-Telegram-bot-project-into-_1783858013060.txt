==================================================
API ENDPOINTS
==================================================

GET /health

Return:

{
    "status":"ok"
}

--------------------------------------------------

GET /v1/models

Return

{
  "object":"list",
  "data":[
      {
          "id":"edge-tts",
          "object":"model"
      }
  ]
}

--------------------------------------------------

GET /v1/audio/voices

Return every available Edge voice.

Each voice should include

Voice ID

Gender

Locale

Language

Friendly Name

Example

{
"id":"en-US-AriaNeural",
"name":"Aria",
"gender":"Female",
"locale":"en-US",
"language":"English"
}

--------------------------------------------------

POST /v1/audio/speech

OpenAI-compatible request body

Example

{
    "model":"edge-tts",
    "input":"Hello World",
    "voice":"en-US-AriaNeural",
    "response_format":"opus"
}
